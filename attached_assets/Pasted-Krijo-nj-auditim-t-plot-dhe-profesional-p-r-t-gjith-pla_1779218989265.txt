Krijo një auditim të plotë dhe profesional për të gjithë platformën LMS/ERP me rolet: Admin, Teacher dhe Student.
Kontrollo çdo pjesë të sistemit për bugs, performance issues, security problems, scalability problems dhe UX issues që mund të shfaqen kur platforma përdoret nga shumë përdorues njëkohësisht.

Kontrolli duhet të përfshijë:

## 1. Authentication & Authorization

* Testo login/logout/reset password
* Kontrollo session handling
* Kontrollo role permissions:

  * Admin nuk duhet të ketë kufizime
  * Teacher nuk duhet të ketë akses në të dhënat e teacherëve tjerë
  * Student nuk duhet të ketë akses në admin routes
* Kontrollo route protection frontend + backend
* Testo token expiration dhe unauthorized requests
* Kontrollo horizontal privilege escalation

## 2. Database & Scalability

* Analizo query performance
* Gjej N+1 queries
* Kontrollo indexes në tables të mëdha
* Simulo:

  * 100+
  * 1,000+
  * 10,000+ users online
* Kontrollo memory leaks
* Kontrollo slow queries
* Kontrollo pagination në:

  * students
  * teachers
  * courses
  * enrollments
  * notifications
  * reports
* Verifiko që asnjë page nuk ngarkon të gjitha records menjëherë
